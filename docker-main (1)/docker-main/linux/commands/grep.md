# 🔹 grep 이란?

* **Global Regular Expression Print** 의 약자
* **파일이나 표준 입력에서 특정 패턴(문자열, 정규표현식 등)을 검색해 출력**하는 명령어
* 원래는 단순 문자열 찾기지만, 정규 표현식을 지원하기 때문에 매우 강력합니다.

---

# 🔹 기본 사용법

```bash
grep [옵션] '패턴' 파일명
```

* `'패턴'` : 찾고 싶은 문자열이나 정규식
* `파일명` : 대상 파일 (생략하면 표준 입력을 검색)

---

# 🔹 주요 예시

### 1) 문자열 검색

```bash
grep "error" /var/log/syslog
```

→ syslog 로그에서 "error"라는 문자열이 포함된 줄 출력

---

### 2) 대소문자 구분 없이 검색

```bash
grep -i "error" /var/log/syslog
```

→ Error, ERROR, eRrOr 도 전부 잡힘

---

### 3) 줄 번호와 함께 출력

```bash
grep -n "main" hello.c
```

→ hello.c 파일에서 "main"이 나온 줄 번호도 표시

---

### 4) 반대로 검색 (해당 문자열이 없는 줄)

```bash
grep -v "INFO" /var/log/syslog
```

→ "INFO"가 없는 줄만 출력

---

### 5) 여러 파일에서 검색

```bash
grep "TODO" *.py
```

→ 현재 디렉터리의 파이썬 파일 전체에서 TODO 검색

---

### 6) 디렉터리 전체를 재귀적으로 검색

```bash
grep -r "password" /etc
```

→ /etc 디렉터리 아래 모든 파일에서 password 포함된 줄 출력

---

### 7) 매칭된 부분만 색깔로 표시

```bash
grep --color=auto "ssh" /var/log/auth.log
```

---

### 8) 프로세스 검색 (자주 쓰는 예)

```bash
ps -ef | grep sshd
```

→ 현재 실행 중인 sshd 프로세스를 검색

---

# 🔹 주요 옵션 정리

| 옵션   | 설명                     |
| ---- | ---------------------- |
| `-i` | 대소문자 무시                |
| `-n` | 줄 번호 표시                |
| `-v` | 패턴이 없는 줄만 표시 (반전)      |
| `-r` | 하위 디렉터리까지 재귀 검색        |
| `-l` | 파일 이름만 표시              |
| `-c` | 매칭된 줄 개수 세기            |
| `-o` | 매칭된 문자열만 출력            |
| `-E` | 정규표현식(ERE) 사용 (확장 정규식) |

---

# 🔹 grep 확장판

* `egrep` = `grep -E` (확장 정규식 지원)
* `fgrep` = `grep -F` (정규식 해석 없이 고정 문자열 검색)

---

✅ **정리:**
`grep`은 리눅스에서 문자열이나 정규표현식으로 텍스트를 검색하는 명령어입니다.
로그 분석, 설정 파일 확인, 프로세스 찾기 등 **일상적인 서버 관리/개발 작업에서 필수 도구**라고 할 수 있어요.
